ccmux は「複数の Claude Code セッションを並列で回す」ために作った軽量 TUI ですが、作っているうちに **vt100 パーサ、PTY 制御、Claude Code の JSONL 解析、Rust の非同期設計** など、書き残しておくと誰かの役に立ちそうなトピックが結構溜まってきました。

このブログはそれを**合間に読める粒度**で書いていく場所です。1 記事 5〜10 分で読めるくらいを目安にしつつ、必要なところは深く掘ります。

## 書こうと思っているテーマ

- **ターミナルエミュレーション**: なぜ ANSI ストリップではなく `vt100` クレートを使うのか
- **PTY の扱い**: `portable-pty` で Windows の ConPTY と Unix の PTY を統一的に使う
- **Claude Code の JSONL を読む**: サブエージェント・TodoList・トークンをライブで拾う
- **Rust + ratatui の描画**: ゼロアロケーション描画と dirty フラグの差分更新
- **スプリットレイアウト**: 二分木で再帰分割する設計

## 読み方

- トップページは新しい記事順
- タグでざっくり領域分け（`rust`・`pty`・`terminal` など）
- RSS も出しているので、好きなリーダーで購読できます

気軽に読んでもらえれば嬉しいです。
