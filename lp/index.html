ccmux-fork

Fork of Shin-sibainu/ccmux with independent development. Installs as ccmux-fork on npm.

This fork periodically syncs upstream changes while developing independent features. For the original project, please visit the upstream repository. 

$ npm install -g ccmux-fork

Migrating from upstream ccmux-cli: 

$ npm uninstall -g ccmux-cli
$ npm install -g ccmux-fork

Differences from upstream

Everything upstream ships with, plus these fork-only additions:

Feature Upstream Fork
Split panes / tabs / file tree / preview ✓ ✓
External IPC control (ccmux list / send / focus / split / new-tab) — ✓
Pane lifecycle events (ccmux events --timeout / --count) — ✓
Screen snapshot API (ccmux inspect) — ✓
IME composition overlay (JP / CN / KR on Windows + third-party terminals) — ✓
Mouse wheel forwarding to alt-screen TUIs (Claude /tui, vim, lazygit) — ✓
Stable IPC error codes + heartbeat half-close detection — ✓
CI matrix (Windows / macOS / Linux) with rustfmt + clippy gate — ✓
